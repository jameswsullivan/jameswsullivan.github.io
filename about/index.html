Networking

Equipment

  • Router: Cisco C2951
  • Switches:
    • Cisco Catalyst 3750G PoE 24
    • Cisco Catalyst 4948-10GE
  • AP:
    • 2 x Ubiquiti UAP-AC-PRO (PoE) in Mesh mode
    • Aerohive/Extreme Networks AP230

Config

  1. VLANS: PROD, DEV, PUBLIC, in their own respective LAN segments. InterVLAN access is controlled by Access Lists.
  2. Catalyst 3750G is configured with InterVLAN Routing using SVI.
  3. C2951 is configured with DHCP pools, DNS, NAT translation, Port Forwarding, and Access Lists. DEV and PROD VLANs have DNS resolution via the DCs for lab use, the PUBLIC VLAN is for a few IoT, streaming, and GUEST WiFi to directly access the Internet.
  4. The Ubiquit APs are configured with two SSIDs in Mesh mode. One SSID taps into the PROD VLAN, the other is tapped into PUBLIC VLAN for GUEST and temporary devices.

      • Nextcloud

      • MySQL databases (for Nextcloud, BookStack, and other apps)
  • Public Environment: for any containers and websites that need to be exposed to the Internet.
